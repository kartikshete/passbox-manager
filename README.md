# 🔐 PassBox Manager

**PassBox Manager** is a simple yet powerful local-first password vault. Built with **React** and **Vite**, it allows you to store and manage your site credentials securely right in your browser.

> No external servers, no tracking, just you and your data. 

### ✨ Key Features
- **Local Storage**: All your passwords stay in your browser's local storage.
- **Auto Favicon**: Automatically fetches favicons for the sites you add.
- **Vault Search**: Quickly find the credentials you need with active filtering.
- **Copy to Clipboard**: One-click copy for usernames and passwords.
- **Theme Support**: Includes both dark and light modes for comfortable browsing.

### 🛠️ Tech Stack
- **Frontend**: [React 19](https://react.dev/), [Vite](https://vitejs.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Styling**: Vanilla CSS with modern custom properties.
- **Routing**: [React Router](https://reactrouter.com/)

### 🚀 Getting Started
```bash
# Clone the repository
git clone https://github.com/kartikshete/passbox-manager.git

# Install dependencies
npm install

# Start development server
npm run dev
```
