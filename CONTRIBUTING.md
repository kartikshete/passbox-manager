# Contributing to PassBox Manager

Thanks for considering contributing! This is a simple project, but I'm always open to improvements.

## How to help:
1. **Reporting Bugs**: Open an issue with steps to reproduce.
2. **Feature Requests**: If you have a cool idea, let me know!
3. **Pull Requests**:
   - Fork the repo.
   - Create a new branch (`git checkout -b feature/cool-feature`).
   - Commit your changes (`git commit -m "added something cool"`).
   - Push to your branch and open a PR.

## Development Setup
- `npm install`
- `npm run dev`

Happy coding!
